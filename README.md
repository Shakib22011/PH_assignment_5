### What is the difference between var, let, and const?
answer:

Difference among var,let and const :<br>
var : var can be redeclared .<br>
let : let cannot be redeclared but updated.<br>
const : const cannot be redeclared or updated<br>

### What is the spread operator (...)?<br>
Answer:

spread operator (...) is an operator that expand elements of an array or object that can be put into another variable as array or object.<br>

### What is the difference between map(), filter(), and forEach()?<br>
Answer: 

map: takes each element of an array and returns a new array.<br>
filter: returns a new array according to a specific condition.<br>
forEach: this runs a function on each element of an array of object or a collection.<br>
