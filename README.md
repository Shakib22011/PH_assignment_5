### What is the difference between var, let, and const?
answer:

Difference among var,let and const :\n
var : var can be redeclared .\n
let : let cannot be redeclared but updated.\n
const : const cannot be redeclared or updated\n

### What is the spread operator (...)?\n
Answer:

spread operator (...) is an operator that expand elements of an array or object that can be put into another variable as array or object.\n

### What is the difference between map(), filter(), and forEach()?\n
Answer: 

map: takes each element of an array and returns a new array.\n
filter: returns a new array according to a specific condition.\n
forEach: this runs a function on each element of an array of object or a collection.\n
