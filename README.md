### What is the spread operator (...)?
Answer:

spread operator (...) is an operator that expand elements of an array or object that can be put into another variable as array or object.

### What is the difference between map(), filter(), and forEach()?
Answer: 

map: takes each element of an array and returns a new array.
filter: returns a new array according to a specific condition.
forEach: this runs a function on each element of an array of object or a collection.
